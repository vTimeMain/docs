vTime Kurulumu

bilgi

vTime'u sunucunuzda sorunsuz kullanabilmek için belirli izinler gerekmektedir. Bunlar olmadan bot sunucunuza eklenemez. Bu izinler şunlardır:

  • Yönetici İzni
  • Sunucuyu Yönet İzni

Adım 1: vTime'u Davet Etmek​

Davet linkini kullanarak sunucunuzu seçip vTime'u sunucunuza davet edin.

Adım 2: Yetkilendirme ve İzin Verme​

vTime'u davet ettikten sonra, hesabınızı yetkilendirmeniz ve botun düzgün çalışması için gerekli izinleri seçmeniz istenecektir.

uyarı

Botun düzgün çalışabilmesi için aşağıdaki izinlere sahip olması gerekmektedir:

  1. VIEW_CHANNEL: Botun kanalları görmesini sağlar.
  2. SEND_MESSAGES: Botun mesaj göndermesini sağlar.
  3. EMBED_LINKS: Botun embed mesajları göndermesini sağlar.
  4. MANAGE_MESSAGES: Botun mesajları düzenlemesini veya silmesini sağlar.

Yardıma mı ihtiyacınız var?​

bilgi

vTime kurulumuyla ilgili daha fazla yardıma ihtiyacınız olursa, Destek sunucumuzdan yardım isteyebilirsiniz. Size yardımcı olmaktan mutluluk duyacağız!
