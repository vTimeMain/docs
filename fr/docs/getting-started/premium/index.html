Premium Bot Information

Bienvenue sur la page d'information de nos bots premium. Sur cette page, vous trouverez une explication complète de notre processus de bot premium, y compris les fonctionnalités disponibles, les niveaux premium, les méthodes de paiement et les questions fréquemment posées.

Premium Niveaux​

Notre plan premium est conçu pour les propriétaires de serveurs qui souhaitent améliorer leurs serveurs avec des fonctionnalités avancées et une protection contre les pirates et les vandales. Notre service premium propose deux niveaux uniques, chacun avec son propre ensemble de fonctionnalités.

Premium Niveau 1​

Le premier niveau est parfait pour les serveurs de taille moyenne qui nécessitent plusieurs fonctions à un coût abordable. Les fonctionnalités incluses dans ce niveau sont:

  • Protection
  • Temporaire Lien
  • Voix en ligne
  • Rôle automatique pour un lien d'invitation spécifique
attention

Veuillez noter que vous ne pouvez pas modifier les préférences du bot et que le lien d'invitation ne peut pas être modifié.

Premium Niveau 2​

Le deuxième niveau est idéal pour les serveurs plus volumineux et comprend toutes les fonctionnalités du niveau 1 Premium, ainsi que:

  • Anti-Raid
  • Bot personnalisé, comprenant un nom d'utilisateur personnalisé, un avatar, un statut et des commandes exclusives.
attention

Veuillez noter que vous ne pouvez pas utiliser de jeton de bot facultatif avec ce niveau.

Gérer Votre Bot Premium​

Vous pouvez gérer votre bot premium depuis une page dédiée. Les options de gestion pour chaque niveau sont les suivantes:

  • Premium Niveau 1: Vous pouvez modifier le cycle de paiement, inviter le bot et le renouveler en utilisant votre mode de paiement préféré.

  • Premium Niveau 2: Vous pouvez modifier la présence, le nom d'utilisateur, l'avatar et le statut du bot, inviter le bot et le renouveler comme avec le niveau 1. De plus, vous aurez un numéro de bot unique que vous pourrez utiliser pour des besoins particuliers.

Cycle de Paiement et Prix​

Vous pouvez vérifier la tarification et le cycle de paiement via le Premium page.

astuce

N'oubliez pas que le forfait annuel est assorti d'une réduction de 50%!

Méthodes de Payement​

Vous pouvez acheter un abonnement premium en utilisant l'une des méthodes suivantes:

  • Credit Cards
  • PayPal
  • Crédits

Pour plus d'informations sur chaque mode de paiement:

  • **Credit Cards**: Nous acceptons toutes les principales credit cards de tous les pays du monde. Le montant affiché sera en dollars.

  • **PayPal**: Assurez-vous que votre compte PayPal est légitime et qu'il contient le montant correct avant d'effectuer un paiement. Si vous rencontrez des difficultés, essayez de lier une credit card.

  • **Crédits**: Notre service fournit une devise de bot appelée Crédits. Vous pouvez charger votre crédit en utilisant différentes méthodes sans dépenser d'argent réel.

Inviter Votre Bot Premium​

Pour accéder aux fonctionnalités premium, vous devez inviter le bot sur votre serveur. Le processus d'invitation peut être complété via la page Gérer Premium, accessible via ce Gérer le Premium.

attention

Dans les rares cas où les fonctionnalités premium restent verrouillées après avoir invité le bot, essayez de désactiver le bot premium ou les bots premium et standard, puis de réinviter le bot premium sur votre serveur. Vous pouvez supprimer le bot standard après avoir acheté la version premium.

FAQ​

Puis-je demander un remboursement?

Oui, vous pouvez. Cependant, vous devez d'abord consulter la politique de remboursement et demander un remboursement dans les sept jours suivant la date d'achat initiale pour une raison raisonnable.

Puis-je avoir plusieurs abonnements sur le même serveur?

Non, vous ne pouvez avoir qu'un seul bot premium par serveur.

Puis-je transférer la propriété de mon bot premium à mon ami?

Oui, vous pouvez! Cependant, cela est limité au plan Niveau 2.

Puis-je annuler le renouvellement automatique?

Oui, vous pouvez en effet! Via la page de gestion de la premium et cliquez sur annuler.
