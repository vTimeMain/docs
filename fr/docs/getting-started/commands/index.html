Commandes

Dans cette section, vous obtiendrez une compréhension plus profonde des commandes de vTime, de leur utilisation et d'un exemple de la façon de mettre de telles commandes en action.

info

  • Comme les commandes de slash sont le préfixe de bot par défaut, tous les usages et exemples seront illustrés à la slash commande /.

  • [ ] Des signes sont requis pour chaque commande référencée avec cet signe, mais ( ) sont facultatifs.

CommandeExplicationUsage

/repAttribuez à quelqu'un un point de réputation. Ne peut être utilisé qu'une fois toutes les 24 heures/rep [utilisateur]
/movemeVous déplace vers un autre salon vocal/moveme [salon ou utilisateur]
/colorChangez votre couleur sur le serveur/color [numéro de couleur]
/colorsVous répertorie toutes les couleurs disponibles/colors
/shortSe raccourcit en URL/short [URL]
/rollLancer des dés/roll (numéro)

Remarques Importantes Sur La Modération

  • Seuls les utilisateurs disposant de l'autorisation Administrateur, Rôles Modérateurs sur le tableau de bord ou de l'autorisation pour cette commande peuvent utiliser les commandes de modération. Par exemple, les utilisateurs disposant d'une autorisation d'interdiction peuvent utiliser la commande "/ban".

  • Si aucune heure n'a été spécifiée, la durée par défaut pour toutes les commandes d'entrée de temps est de 3 mois; celui-ci peut être modifié à tout moment. Pour plus d'informations, veuillez visiter [modération] ().
