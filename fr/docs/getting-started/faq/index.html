Foire Aux Questions

Cette section FAQ est conçue pour fournir des réponses aux questions courantes que les utilisateurs ont à propos du bot. La FAQ est divisée en trois catégories principales :

  • FAQ Générales
  • FAQ sur la Modération
  • FAQ sur le Premium

FAQ Générales​

Puis-je utiliser vTime gratuitement ?

Oui, vTime peut être utilisé gratuitement. Cependant, certaines fonctionnalités peuvent nécessiter un abonnement premium pour y accéder. Vous pouvez consulter les tarifs et les options d'abonnement sur la Page Premium.

Pourquoi le tableau de bord affiche-t-il tous les serveurs dans lesquels je suis ?

vTime affiche tous les serveurs auxquels vous appartenez. Vous pouvez facilement inviter le bot sur n'importe quel serveur où vous avez les permissions nécessaires. Si vous n'avez pas les permissions requises, le bot vous en informera.

Existe-t-il un moyen de signaler des bugs ou de suggérer de nouvelles fonctionnalités pour vTime ?

Oui, vous pouvez signaler des bugs ou suggérer de nouvelles fonctionnalités en rejoignant le serveur officiel de vTime Support, où vous pouvez partager vos commentaires avec les développeurs et les autres membres de la communauté.

Les membres bloqués sont-ils bloqués de la liste des 3 meilleurs aujourd'hui ?

Oui, si un membre est bloqué, il sera bloqué de la liste des 3 meilleurs du serveur et sera également exclu des calculs généraux.

FAQ sur la Modération​

Quels sont les permissions nécessaires pour que mes utilisateurs puissent utiliser les commandes de modération ?

ADMINISTRATOR et MANAGE_GUILD ou MANAGE_SERVER

Puis-je donner l'accès aux utilisateurs pour utiliser les commandes de modération sans leur donner de permissions ?

Oui, vous pouvez créer un rôle de modérateur sur votre serveur même si ce rôle n'a pas de permissions, puis allez dans le tableau de bord > Actions de modération > Gérer la modération. À partir de là, sélectionnez les rôles de modération. Les utilisateurs affectés à ces rôles pourront gérer le tableau de bord et utiliser les commandes de modération.

FAQ sur le Premium​

Comment obtenir l'abonnement premium ?

Actuellement, les fonctionnalités premium sont fournies gratuitement à tous les utilisateurs dans le cadre de la version BETA.
