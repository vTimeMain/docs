Tableau de Bord : Gestion du Serveur

De nombreuses fonctionnalités sont disponibles, toutes facilement accessibles depuis le tableau de bord. Vous pouvez configurer les paramètres, attribuer des rôles de modérateur pour utiliser les commandes de modération, et bien plus encore.

Accéder au Tableau de Bord​

L'accès au tableau de bord est simple et rapide. Voici les étapes à suivre :

  • Rendez-vous sur le site web et connectez-vous à votre compte.
  • Le bouton du tableau de bord apparaîtra sur la même page, cliquez dessus pour y accéder.
attention

Vous devez posséder l’une des permissions suivantes pour modifier votre serveur :

  1. ADMINISTRATOR
  2. MANAGE_GUILD (ou MANAGE_SERVER)
  3. Un rôle de modérateur personnalisé

Configuration des Modules et Commandes​

Certains modules du tableau de bord disposent d'un bouton d'activation qui permet de les activer ou de les désactiver. De plus, certaines pages comportent un bouton en haut permettant de désactiver toutes les options de la page. Il existe également un bouton d'édition/de configuration pour personnaliser ces options.

Voici quelques options de personnalisation disponibles :

  • Paramètres du Serveur : Réinitialise les paramètres par défaut. Cela affectera uniquement les paramètres suivants : NotifyChannel (Désactivé), PrivateNotify (Désactivé), langue (en), modRoles (vide), UseCommandsChannel (Désactivé) et CommandsChannel (vide).
  • Langue : Permet de changer la langue du bot.
  • Gérer les Commandes : Si vous activez l’interrupteur et sélectionnez un ou plusieurs canaux, le bot acceptera les commandes uniquement dans les canaux sélectionnés.
  • Gérer la Liste Noire : Cette section affiche en temps réel les membres bloqués. Vous pouvez retirer un membre bloqué en cliquant sur le bouton "Supprimer". Pour bloquer un nouveau membre, cliquez sur le bouton "Bloquer", saisissez la raison et recherchez le membre par son ID, son nom, son tag ou son nom d’utilisateur. Le membre sera bloqué immédiatement.
  • Gérer la Modération : Sélectionnez un rôle de modérateur pour accorder l’accès aux paramètres et commandes de modération.
attention

Vous ne voyez pas votre serveur dans la liste ? Cela signifie que vous n’avez pas les permissions nécessaires pour accéder pleinement au Tableau de Bord, telles que Administrator, MANAGE_GUILD ou un rôle de modérateur personnalisé. Assurez-vous que votre rôle dispose bien des permissions requises dans les paramètres des rôles.
