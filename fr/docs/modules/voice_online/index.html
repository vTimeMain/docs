Voix en Ligne

Une fonctionnalité qui permet aux utilisateurs de suivre le nombre de membres activement connectés aux canaux vocaux du serveur.

attention

Cette fonctionnalité n'est disponible que pour les abonnés du Premium niveau 1 et 2.

info

Le tableau de bord est l'endroit où les configurations décrites ici sont gérées.

  1. Accédez au tableau de bord et connectez-vous.
  2. Sélectionnez votre serveur dans le menu de gauche.
  3. Choisissez le module et assurez-vous qu'il est activé.

Texte​

Le texte que vous souhaitez que la pièce ou la catégorie affiche et mette à jour. [00] est la variable pour le nombre de membres, elle doit être ajoutée pour que le bot affiche le nombre d'utilisateurs activement connectés aux salons vocaux. Cela permet aux utilisateurs de voir le nombre de membres en temps réel dans les salons vocaux, ce qui permet de voir facilement combien de personnes sont actuellement actives et participent.

Salon​

Le salon sur lequel vous souhaitez afficher le texte, gardez à l'esprit qu'il peut s'agir d'un salon vocal ou d'une catégorie. Cette fonctionnalité permet aux utilisateurs de choisir le salon dans lequel le texte du nombre de membres sera affiché, leur donnant la possibilité de choisir le meilleur emplacement pour leur serveur.

Salons Cachées

Cette option vous donnera la possibilité d'empêcher le bot de compter les membres connectés à un salon spécifique, où ils ne seront pas inclus dans le décompte.

Pas de Bots​

Il s'agit d'une option facultative, si elle n'est pas activée, la fonctionnalité inclura le nombre de bots connectés au système vocal en ligne. S'il est activé, le nombre de bots ne sera pas inclus. Cette fonctionnalité est utile pour les utilisateurs qui souhaitent voir le nombre d'utilisateurs humains dans les salons vocaux, plutôt que le nombre total d'utilisateurs, y compris les bots.

attention

Veuillez noter que le nombre de membres sera mis à jour toutes les 5 minutes.

Dans l'ensemble, Voice Online est une fonctionnalité puissante qui permet aux utilisateurs de suivre le nombre de membres dans les salons vocaux de leur serveur, leur fournissant des informations en temps réel et leur donnant la possibilité de choisir le meilleur emplacement pour le texte du nombre de membres. Il offre également des options pour masquer certains salons du nombre et pour exclure le nombre de bots du nombre total de membres, ce qui en fait un outil polyvalent pour gérer et surveiller un serveur.
