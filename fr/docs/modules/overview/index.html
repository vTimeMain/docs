Vue d'ensemble

Affiche les 3 meilleurs membres, les 3 meilleurs membres du jour et les 3 derniers changements dans les positions du top 10.

remarque

Ces fonctionnalités sont premium et peuvent être activées après avoir activé le canal de notifications.

Top 3 Temps de Voix
Top 3 Aujourd'hui

Assurez-vous que l'interrupteur de notification est activé.

info
  • Le temps des 3 meilleurs membres sera mis à jour toutes les 10 secondes en temps réel.
  • Les 3 derniers changements seront mis à jour directement dès que les changements se produiront.
  • Si le top 3 est verrouillé, assurez-vous que l'interrupteur de notification est activé. Vous pouvez aller dans 'Notifications Top 10', puis activer l'interrupteur et activer l'interrupteur du canal de notifications. Ensuite, sélectionnez le canal où les nouvelles notifications seront envoyées, et les Top 3 fonctionneront sur la page Vue d'ensemble.
