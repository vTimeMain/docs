Journaux Premium

Activez les journaux pour commencer à enregistrer immédiatement toutes les actions de vTime.

info

Le tableau de bord est l'endroit où les configurations décrites ici sont gérées.

  1. Allez dans le tableau de bord et connectez-vous.
  2. Sélectionnez votre serveur dans le menu à gauche.
  3. Sélectionnez la section Journaux et commencez à configurer vos canaux de journaux, en les personnalisant selon vos besoins.

Paramètres​

Chaque journal contient un interrupteur, un canal de journalisation et une couleur d'intégration.

  • Temps de blocage du membre
  • Temps de déblocage du membre
  • Réinitialiser le temps du membre
  • Réinitialiser le temps de tous les membres
  • Changer la langue
remarque

Assurez-vous que l'interrupteur du haut est activé pour commencer à enregistrer les journaux.
