Команды

В этом разделе вы лучше поймёте команды vTime, их применение и примеры использования.

предупреждение

  • Так как слеш является префиксом бота по умолчанию, все примеры использования будут на примере слеш-команд /.

  • Знаки [ ] обязательны для каждой команды, на которую ссылается этот знак, а ( ) необязательны.

Общее​

КомандаОписаниеИспользование
/creditsПоказать ваши или чьи-либо кредиты/credits [пользователь] (число)
/repНаградить кого-нибудь очком репутации. Можно использовать раз в 24 часа/rep [юзер]
/movemeПеремещает вас в другой голосовой канал/moveme [канал или юзер]
/colorСменить ваш цвет на сервере/color [номер цвета]
/colorsПоказывает список доступных цветов/colors
/shortСокращает URL/short [URL]
/rollБросок игральных костей/roll (число)

Система уровней​

КомандаОписаниеИспользование
/profileПосмотреть ваш или чей-либо кастомизируемый персональный общедоступный профиль/profile (юзер)
/rankПосмотреть вашу или чью-либо рейтинговую карточку на сервере/rank (юзер)

/setlevelУстанавливает уровень юзера/setlevel [юзер][тип] [новый_уровень]

Важные примечания по модерации

  • Только пользователи с разрешениями Администратор, Роли модераторов в панели управления, или с разрешением на эту команду могут использовать команды модерации. Например, пользователи с разрешением на бан могут использовать команду /ban.

  • Если время не указано, то время по умолчанию для всех команд с указанием времени — 3 месяца; это может быть изменено в любое время.
