Frequently Asked Questions

This FAQ section is designed to provide answers to common questions that users have about the bot. The FAQ is divided into three main categories:

  • General FAQ
  • Moderation FAQ
  • Premium FAQ

General FAQ​

Can I use vTime for free?

Yes, vTime can be used for free. However, some features may require a premium subscription to access. You can check the pricing and subscription options on the Premium Page.

Why does the dashboard display all the servers I'm in?

vTime displays all the servers you are a member of. You can easily invite the bot to any server where you have the necessary permissions. If you don’t have the required permissions, the bot will notify you.

Is there a way to report bugs or suggest new features for vTime?

Yes, you can report bugs or suggest new features by joining the official vTime Support server, where you can share your feedback with the developers and other members of the community.

Are blocked members blocked from the Top Today list?

Yes, if a member is blocked, they will be blocked from the server's Top 3 list and excluded from general calculations as well.

Moderation FAQ​

What permissions do my users need to use the moderation commands?

ADMINISTRATOR and MANAGE_GUILD or MANAGE_SERVER

Can I give users access to use moderation commands without giving them permission?

Yes, you can create a moderator role in your server even if the role has no permission and then go to Dashboard > Mod Actions > Manage Moderation. From there, select the moderation roles. Users assigned to these roles will be able to manage the dashboard and use moderation commands

Premium FAQ​

How can I get the premium subscription?

Currently, the premium features are provided for free to all users as part of the BETA version!
