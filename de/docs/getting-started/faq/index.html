Frequently Asked Questions

This FAQ section is designed to provide answers to common questions that users have about the bot. The FAQ is divided into three main categories:

  • General FAQ
  • Moderation FAQ
  • Premium FAQ

General FAQ​

Can I use vTime for free?

Yes, vTime can be used for free. However, some features may require a premium subscription to access. You can check the pricing and subscription options on the Premium Page.

Can I change the bot's prefix?

Unfortunately, the bot no longer uses prefixes. Instead, we have implemented the use of discord slash commands. However, users can set aliases for their favorite commands and use them as an alternative.

Can I change the bot's timezone?

At this time, it is not possible to change the bot's timezone. The default timezone is set to UTC+3.

Can I give specific users access to dashboard?

Only users with the Administrator permission can have access to the dashboard. To grant access to specific users, create a role and assign it to them.

Is there a way to report bugs or suggest new features for vTime?

Yes, you can report bugs or suggest new features by joining the official vTime Support server, where you can share your feedback with the developers and other members of the community.

Moderation FAQ​

What is the default duration for actions?

If no time is specified, the default duration for actions is three months.

What permissions do my users need to use the moderation commands?

The permissions required for each moderation command vary. To determine the necessary permission, users can try to use the command and receive an error message indicating the required permission.

Can I give users access to use moderation commands without giving them permission?

Yes, it is possible to give users access to use moderation commands without giving them permission. The bot has an option called Mod Roles which allows users to specify roles that are allowed to use all moderation commands.

Premium FAQ​

How can I cancel my premium subscription?

You can cancel your premium subscription by going to the manage premium page on the bot's website and selecting the Edit option for the desired bot, then Cancel Subscription. Once the subscription is cancelled, you will no longer be charged and your premium features will be deactivated once the bot's duration expires.

What happens if I downgrade from premium to free version?

If you downgrade from the premium version to the free version, you will lose access to all premium features. Your data will not be deleted, but it will be locked until you upgrade again.

Is there a free trial for the premium version?

Unfortunately, there is no free trial for the premium version.

Do I get a refund if I am not satisfied with the premium version?

Refunds will be considered on a case-by-case basis. If you would like to request a refund, please contact our Support Team with the reason for your request.

How often are new premium features added?

New premium features are added regularly, but there is no set schedule for when they will be added. To stay informed about new features and updates, you can join the official vTime Support server, where developers and members of the community share news and information about the bot.
