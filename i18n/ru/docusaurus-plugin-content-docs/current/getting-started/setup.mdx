# Настройка vTime

:::info

Для того, чтобы пригласить vTime на ваш сервер, необходимы определенные разрешения. Без них бот не может быть добавлен на ваш сервер. К этим разрешениям относятся:

- Право `Администратор`
- Право `Управление сервером`

:::

## Шаг 1: Приглашение vTime

Воспользуйтесь [инвайт-ссылкой](https://vTime.io/invite) для того, чтобы пригласить vTime на ваш сервер и авторизуйте ваш аккаунт, выбрав свой сервер.

## Шаг 2: Авторизация и предоставление разрешений

Во время приглашения vTime вам будет предложено авторизовать аккаунт и выбрать необходимые разрешения для правильной работы бота.

## Шаг 3: Настройка роли vTime

Для того, чтобы увидеть роли сервера в некоторых модулях панели управления, переместите роль vTime выше остальных ролей. Для этого:

- Перейдите в настройки сервера
- Выберите Роли
- Зажмите и переместите роль vTime выше всех

:::caution

Discord не всегда отображает роли в панели управления, даже когда роль vTime выше всех. Если это происходит, попробуйте эти шаги:

1. Опустите роль vTime
2. Сохраните изменения
3. Поднимите роль vTime
4. Снова сохраните изменения

:::

## Нужна помощь?

:::info

Если вам нужна дополнительная помощь в настройке vTime, присоединитесь к [Серверу поддержки](https://discord.gg/vTime) и попросите о помощи. Наша команда будет рада помочь вам!

:::
