# Введение

vTime — это очень настраиваемый многоцелевой бот, который предлагает такие функции, как приветственные изображения, детализированные логи, социальные команды, модерацию, роли самостоятельного назначения, уведомления с социальных сетей и защиту для вашего сервера.

## О нашей документации

Наша документация содержит полные объяснения по использованию и настройке каждой функции vTime, а также советы по устранению неполадок, которые могут возникнуть.

## Модули

С лёгкостью переключайтесь к нужному модулю, кликнув на его имя в левом боковом меню. Каждый модуль имеет собственную страницу конфигурации.

## Команды

Все команды vTime имеют специальную страницу с примерами использования и информацией. Для получения более подробной информации посетите нашу [страницу команд](https://vTime.io/commands).

## Оформление

В данной документации используется следующая система цветового кодирования:

:::tip
Указывает на советы, связанные с функцией.
:::

:::info
Предоставляет информацию, связанную с функцией.
:::

:::caution
Основные предупреждения, связанные с функцией.
:::

:::note
Показывает примечания, связанные с функцией.
:::

:::danger
Указывает на известные ошибки, проблемы и то, чего следует избегать, что может негативно повлиять на рабочий процесс.
:::
