# Логи

Поручите ProBot немедленно и чётко логировать все действия сервера.

:::info

Панель управления — это место, где осуществляется управление описанными здесь конфигурациями.

1. Перейдите в панель управления и авторизуйтесь. 
2. Выберите ваш сервер в левом меню.
3. Выберите модуль и убедитесь, что он включён. 

:::

## Настройки

Каждый лог будет содержать переключатель, канал для логирования и цвет embed-сообщения.

:::important 

Некоторые логи могут требовать Премиум 2 уровня по причинам производительности.

:::

- Участник забанен



- Тайм-аут



- Канал создан



- Роль создана



- Канал удалён



- Сообщение удалено



- Роль удалена



- Сообщение отредактировано



- Участник кикнут



- Участник перемещён в другой голосовой канал



- Участник отключён от голосового канала


- Участник присоединился



- Участник вышел



- Никнейм изменён



- Применена команда модерации



- Роль выдана



- Роль снята



- Приглашения сервера



- Участник разбанен



- Канал обновлён



- Обновлены разрешения канала



- Роль обновлена



- Изменения сервера



- Участник присоединился к войсу



- Участник покинул войс



- Состояние войса



- Участник переключился между голосовыми каналами


:::caution

Иногда бот может не логировать действия из-за проблем с правами; если это произошло, кикните и повторно пригласите бота и убедитесь, что у него есть право `Управлять вебхуками`.

:::
