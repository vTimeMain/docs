# Messages en Embed

Les Messages en Embed enrichie permet à votre bot d'envoyer des versions personnalisées de vos messages préférés. Ce guide vous guidera tout au long du processus de configuration et de configuration de ces messages en embed.

:::Info

Le tableau de bord est l'endroit où les configurations décrites ici sont gérées.

1. Accédez au tableau de bord et connectez-vous.
2. Sélectionnez votre serveur dans le menu de gauche.
3. Choisissez le module et assurez-vous qu'il est activé.

:::

## Création d'une Nouvelle Embed

Pour créer une nouvelle embed, cliquez sur le bouton "Créer". Cela vous redirigera vers une nouvelle page où vous pourrez prévisualiser l'apparence du messages en embed et effectuer les ajustements nécessaires. Cette page contient également plusieurs boutons:

- Envoyer à un Salon
- Modifier un Message Envoyé
- Enregistrer les Modifications
- Annuler les Modifications

## Configuration du Message en Embed

Sur la nouvelle page, vous verrez un bouton supplémentaire intitulé `Ajouter des champs` qui contient toutes les options de personnalisation de votre message en embed. Ces options incluent:

- Titre
- Description
- URL
- Image
- Miniature
- Auteur
- Couleur
- Pied de page
- Champ

Après avoir sélectionné les options souhaitées, une zone de texte vous sera présentée dans laquelle vous pourrez saisir le texte de votre message.

## Envoi la Embed à un Salon

Une fois que vous avez configuré votre message de embed, vous pouvez utiliser le bouton `Envoyer à un salon` pour l'envoyer au salon souhaité.

## Enregistrer les Modifications

Pour enregistrer les modifications apportées à votre messages en embed, utilisez le bouton `Enregistrer les modifications`.

## Modifier d'un Message Envoyé

Si vous avez besoin de modifier un message existant qui a déjà été envoyé, vous pouvez utiliser le bouton `Modifier le message`.

## Annuler des Modifications

Si vous devez annuler les modifications apportées à votre message en embed, utilisez le bouton `Annuler les modifications`

:::caution

Veuillez noter qu'il y a des limites à la fonction de embed de Discord, donc si le bot n'est pas en mesure d'envoyer le messages en embed, vous avez peut-être dépassé la limite. Pour éviter cela, assurez-vous de remplir complètement tous les champs obligatoires et faites attention aux limites suivantes:

- Le titre embed est limité à 256 caractères
- La description embed est limitée à `4096` caractères
- Une embed peut contenir un maximum de `25` champs
- Un nom/titre de champ est limité à `256` caractères et la valeur du champ est limitée à `1024` caractères
- Le pied de page embed est limité à `2048` caractères
- Le nom de l'auteur embed est limité à `256` caractères
- Le nombre total de caractères autorisés dans une embed est de `6000`

:::

En respectant ces limites, vous pouvez vous assurer que votre bot est capable d'envoyer le message d'intégration sans aucun problème.
