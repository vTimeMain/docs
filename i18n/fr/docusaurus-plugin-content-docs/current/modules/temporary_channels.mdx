# Salons Temporaires: une fonctionnalité ProBot pour les canaux contrôlés par l'utilisateur

Les salons temporaires sont une fonctionnalité intéressante de ProBot qui permet aux utilisateurs de créer et de gérer leurs propres salons temporaires sur un serveur. Grâce à cette fonctionnalité, les utilisateurs peuvent créer et personnaliser leurs propres salons comme ils l'entendent, avec un ensemble d'options personnalisées que les propriétaires de serveurs peuvent personnaliser pour répondre à leurs besoins. Dans ce guide, nous passerons en revue les étapes de configuration des salons temporaires sur votre serveur ProBot, y compris la création d'une catégorie et la configuration des options disponibles pour les utilisateurs.

:::info

Le tableau de bord est l'endroit où les configurations décrites ici sont gérées.

1. Accédez au tableau de bord et connectez-vous.
2. Sélectionnez votre serveur dans le menu de gauche.
3. Choisissez le module et assurez-vous qu'il est activé.
:::

## Configuration

### Configuration de la Catégorie

Pour commencer, vous devrez créer une catégorie spéciale pour vos salons temporaires, c'est là que tous les salons temporaires seront situés.

### Créer une Salon

Pour utiliser cette fonctionnalité, les membres doivent d'abord rejoindre un canal vocal désigné, vous devrez donc créer un salon vocal. Une fois qu'ils sont dans ce canal, ils peuvent alors créer leurs propres salons temporaires.

:::info

Vous pouvez configurer les options suivantes pour répondre aux besoins de votre serveur:

- **Temps de suppression de salon**: il s'agit de la durée pendant laquelle une salon temporaire peut rester vide avant d'être automatiquement supprimée. Cette option est personnalisable et peut être définie sur n'importe quelle valeur de votre choix.

- **Salons maximum par utilisateur**: il s'agit du nombre maximum de salons temporaires qu'un utilisateur peut créer à la fois. Cette option est configurée pour empêcher le spam de salons, et elle est personnalisable afin que vous puissiez définir le nombre maximum de salon avec lesquelles vous êtes à l'aise.

- **Maximum d'utilisateurs par salon**: il s'agit du nombre maximal d'utilisateurs pouvant rejoindre un salon temporaire en même temps. Cette option est personnalisable afin que vous puissiez définir le nombre maximum d'utilisateurs avec lesquels vous êtes à l'aise.

:::

:::caution

Les utilisateurs disposant de l'autorisation `Move Members` contourneront la limite d'utilisateurs.

:::

Avec ces personnalisations en place, vos membres pourront créer leurs propres salons temporaires avec les paramètres souhaités, et vous pourrez les gérer facilement.
