# Actions Éffectués

Demandez à ProBot de consigner immédiatement et proprement toutes les actions du serveur.

:::info

Le tableau de bord est l'endroit où les configurations décrites ici sont gérées.

1. Accédez au tableau de bord et connectez-vous.
2. Sélectionnez votre serveur dans le menu de gauche.
3. Choisissez le module et assurez-vous qu'il est activé.

:::

## Paramètres

Chaque action éffectué contiendra un interrupteur à basculè, un salon,et une couleur d'intégration.

:::important

Certains actions éffectués peuvent nécessiter un stockage premium de niveau 2 pour des raisons de performances.

:::

- Membre banni



- Temps déduit



- Salon créé



- Rôle créé



- Salon supprimé



- Message supprimé



- Rôle supprimé



- Message modifié



- Membre expulsé



- Membre déplacé dans un autre salon vocal



- Membre déconnecté d'un salon vocal



- Arrivée d'un membre



- Départ d'un membre



- Surnom changé



- Commande de modération utilisée



- Rôle donné



- Rôle supprimé



- Invitations de serveur



- Membre débanni



- Salon mis à jour



- Mise à jour des permissions des salons



- Mise à jour des rôles



- Mettre à jour le serveur



- Connexion d'un membre à un salon vocal



- Déconnexion d'un membre d'un salon vocal



- État de la voix



- Membre déplacé entre les salons vocaux




::caution

Parfois, le bot ne peut pas enregistrer les actions dans les journaux en raison d'un problème d'autorisations; si cela se produit,veuillez lancez et réinvitez le bot et assurez-vous qu'il dispose l'autorisations `Manage webhooks`.

:::
