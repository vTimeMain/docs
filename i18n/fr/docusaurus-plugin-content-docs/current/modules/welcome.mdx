# Messages de Bienvenue et Départ Personnalisés pour Engager la Gestion du Serveur Discord

Les messages de bienvenue et Départ jouent un rôle essentiel dans l'établissement d'une communauté accueillante et engageante sur votre serveur Discord. Ces messages peuvent faire en sorte qu'un nouveau membre se sente valorisé et lui donner un sentiment d'appartenance. De même, lorsqu'un membre quitte le serveur, un message de départ bien conçu peut créer une impression positive et montrer que le serveur apprécie la contribution du membre.

Ce guide fournit un aperçu complet des étapes nécessaires pour configurer les messages de bienvenue et départ sur votre serveur Discord.

:::info

Le tableau de bord est l'endroit où les configurations décrites ici sont gérées.

1. Accédez au tableau de bord et connectez-vous.
2. Sélectionnez votre serveur dans le menu de gauche.
3. Choisissez le module et assurez-vous qu'il est activé.

:::

## Configuration des Messages de Bienvenue

### Envoi un Message

Un message de bienvenue est un excellent moyen de saluer les nouveaux membres et de les faire se sentir valorisés. Pour configurer les messages de bienvenue, procédez comme suit:

1. Activez l'option pour les messages de bienvenue dans le tableau de bord.

2. Saisissez le texte que vous souhaitez envoyer aux nouveaux membres lorsqu'ils rejoignent le serveur.

3. Choisissez d'envoyer le message sous forme de message direct au nouveau membre ou à un canal spécifique sur le serveur.

### Envoi une Image

L'envoi d'une image de bienvenue est un autre moyen de créer une atmosphère chaleureuse et invitante pour les nouveaux membres. Pour configurer les images de bienvenue, procédez comme suit:

1. Activez l'option pour les images de bienvenue dans le tableau de bord.

2. Choisissez l'une des trois options disponibles pour envoyer l'image:

- AVEC MESSAGE TEXTE
- AVANT LE MESSAGE TEXTE
- VERS UN SALON

Personnalisez l'image de bienvenue à l'aide des options disponibles:

- **Arrière-plan:** Choisissez soit un arrière-plan transparent, soit une image avec une largeur et une hauteur personnalisables.
- **Avatar:** Choisissez entre un avatar carré ou circulaire avec une hauteur, une largeur et des coordonnées personnalisables.
- **Nom d'utilisateur:** Choisissez la position, la taille, la couleur et l'alignement du nom d'utilisateur de l'utilisateur sur l'image de bienvenue.
- **Texte:** Saisissez le texte que vous souhaitez afficher sur l'image d'accueil, ainsi que sa position, sa taille et sa couleur.

:::caution

Les images d'arrière-plan doivent être inférieures à 3 Mo et les types d'images pris en charge sont (`jpg, png et jpeg`). gif ne sont actuellement pas pris en charge.

:::

## Configuration des Messages de Départ

Un message de départ est un excellent moyen de montrer votre appréciation aux membres qui ont quitté votre serveur. Pour configurer les messages d'adieu, procédez comme suit:

1. Activez l'option pour les messages de départ dans le tableau de bord.
2. Saisissez le texte que vous souhaitez envoyer aux membres lorsqu'ils quittent le serveur.
3. Choisissez le canal auquel vous souhaitez que le message d'adieu soit envoyé.

## Informations Complémentaires

:::Info

- Chaque boîte de message texte a une variable utilisable que vous pouvez insérer dans le texte de votre message, comme le nom d'utilisateur de l'utilisateur ou le nom du serveur. Cette fonctionnalité peut aider à personnaliser les messages de bienvenue et départ et à les rendre plus engageants.

- La personnalisation des messages de bienvenue et départ est un excellent moyen de mettre en valeur la personnalité de votre serveur et de le démarquer des autres serveurs. Envisagez d'expérimenter différents messages et images pour trouver ce qui fonctionne le mieux pour votre communauté.

:::
