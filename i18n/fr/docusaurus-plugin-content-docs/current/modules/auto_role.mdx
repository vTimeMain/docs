# Rôles Automatiques

La fonction Rôles Automatiques de ProBot lui permet d'attribuer automatiquement des rôles aux utilisateurs nouvellement rejoints sur leur serveur Discord. Ce guide vous guidera à travers les étapes d'installation et de configuration de cette fonctionnalité.

:::Info

Le tableau de bord est l'endroit où les configurations décrites ici sont gérées.

1. Accédez au tableau de bord et connectez-vous.
2. Sélectionnez votre serveur dans le menu de gauche.
3. Choisissez le module et assurez-vous qu'il est activé.

:::

## Configuration

Une fois que vous avez accédé au module Rôles Automatiques, vous pouvez commencer à configurer les rôles qui seront automatiquement attribués aux nouveaux utilisateurs. Pour ce faire, utilisez le menu de sélection du module pour choisir un ou plusieurs rôles selon vos besoins.

## Rôle Automatique Premium

Vous pouvez définir un lien unique pour les rôles automatiques et particuliers si vous êtes un abonné premium.

1. Sélectionnez `Ajouter`.
2. Ensuite, cliquez sur le bouton `Modifier`.
3. Insérez le lien d'invitation.
4. Sélectionnez les rôles que vous voudriez que les membres aient s'ils rejoignent via ce lien d'invitation.

:::caution

1. Le rôle du bot doit être supérieur au rôle automatique choisi.

2. Si le panneau de l'écran des règles Discord est activé, les utilisateurs nouvellement rejoints devront accepter les règles avant de recevoir le rôle automatique.

3. Le bot n'attribuera pas de rôles aux utilisateurs avec des privilèges administratifs.

:::

En suivant ces étapes et en prenant note des notes importantes, vous pouvez installer et configurer avec succès la fonction Auto-Roles sur votre serveur Discord à l'aide de ProBot.
