# Configurer vTime

:::info

Afin d'inviter vTime sur votre serveur, certaines permissions sont requises. Sans ceux-ci, le bot ne peut pas être ajouté à votre serveur. Ces permissions sont:

- `Administrator` Permission
- `Manage Servers` Permission

:::

## Étape 1: Inviter vTime

Utilisez le [lien d'invitation](https://vTime.io/invite) pour inviter vTime sur votre serveur et autoriser votre compte, en sélectionnant votre serveur.

## Étape 2: Autoriser et Accorder des Autorisations

Après avoir invité vTime, vous serez invité à autoriser votre compte et à sélectionner les permissions nécessaires pour que le bot fonctionne correctement.

## Étape 3: Configurer le Rôle de vTime

Pour afficher les rôles de serveur dans certains modules de tableau de bord, déplacez le rôle de vTime en haut de la hiérarchie des rôles de serveur. Pour faire ça:

- Accédez aux paramètres du serveur
- Cliquez sur les rôles
- Maintenez-cliquez et faites glisser le rôle de vTime en haut de la liste

:::caution

Discord n'affiche pas toujours les rôles sur le tableau de bord, même après avoir fait glisser le rôle de vTime vers le haut. Si cela se produit, essayez ces étapes:

1. Faites glisser le rôle de vTime vers le bas
2. Enregistrez les modifications
3. Faites glisser le rôle de vTime vers le haut
4. Enregistrez à nouveau les modifications

:::
