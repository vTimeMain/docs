# Comandos

En esta sección, obtendrá una comprensión más profunda de los comandos ProBot, su uso y un ejemplo de cómo poner dichos comandos en acción.

:::important

- Como los comandos de barra son el prefijo bot por defecto, todos los usos y ejemplos se ejemplificarán con el comando de barra `/`.

- Los signos `[` `]` son obligatorios para cada orden referenciada con ese signo, pero `(` `)` son opcionales.

:::

## General

| Comando  |                                      Explicación                                      |              Uso              |
| -------- | :-----------------------------------------------------------------------------------: | :---------------------------: |
| /credits |                        Muestre su crédito o el de otra persona                        | /credits [usuario] (cantidad) |
| /rep     | Concede a alguien un punto de reputación. Sólo puede utilizarse una vez cada 24 horas |        /rep [usuario]         |
| /moveme  |                            Te desplaza a otro canal de voz                            |   /moveme [canal o usuario]   |
| /color   |                            Cambia tu color en el servidor                             |   /color [número de color]    |
| /colors  |                          Lista todos los colores disponibles                          |            /colors            |
| /short   |                                    Acorta una URL                                     |         /short [URL]          |
| /roll    |                                     lanzar dados                                      |        /roll (número)         |

<details>

<summary>Notas importantes sobre la moderación</summary>

- Sólo los usuarios con permiso de `Administrador`, `Mod Roles` en el dashboard, o el permiso para ese comando pueden usar comandos de moderación. **Por ejemplo**, los usuarios con permiso de baneo pueden usar el comando `/ban`.

- Si no se ha especificado ningún tiempo, la duración por defecto para todos los comandos de entrada de tiempo es de **3 meses**; esto puede modificarse en cualquier momento.

</details>
