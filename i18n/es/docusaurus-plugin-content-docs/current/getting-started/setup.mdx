# Configurar ProBot

:::info

Para invitar a ProBot a tu servidor, se requieren ciertos permisos. Sin ellos, el bot no podrá ser añadido a tu servidor. Estos permisos son:

- `Administrador` Permiso
- `Gestionar Servidor` Permiso

:::

## Paso 1: Invitar a ProBot

Utilice el [enlace de invitación](https://probot.io/invite) para invitar a ProBot a su servidor y autorizar su cuenta, seleccionando su servidor.

## Paso 2: Autorización y Concesión de permisos

Al invitar a ProBot, se te pedirá que autorices tu cuenta y selecciones los permisos necesarios para que el bot funcione correctamente.

## Paso 3: Establecer el rol de ProBot

Para mostrar los roles de servidor en algunos apartados del dashboard, mueva el rol de ProBot a la parte superior de todos los roles de servidor. Para ello

- Ir a la configuración del servidor
- Haga clic en roles
- Mantenga pulsado y arrastre el rol de ProBot a la parte superior de la lista

:::caution

Es posible que Discord no siempre muestre los roles en el dashboard, incluso después de arrastrar el rol de ProBot a la parte superior. Si esto ocurre, prueba estos pasos:

1. Arrastra el rol de ProBot hacia abajo
2. Guarde los cambios
3. Arrastra el rol de ProBot hacia arriba
4. Vuelva a guardar los cambios

:::
