# はじめに

vTime は、高度にカスタマイズ可能な多機能 Bot で、画像付きのウェルカムメッセージ、詳細なログ、便利なコマンド、モデレーション、リアクションロール、SNS 通知、サーバー保護などの機能を提供します。

## ドキュメントについて

当社のドキュメントでは、vTime の各機能の使用方法と設定方法、そして問題が発生した場合のトラブルシューティングのヒントについて包括的に説明しています。

## モジュール

左側のサイドバーにあるモジュール名をクリックすると、簡単に特定のモジュールに移動できます。各モジュールには独自の設定ページがあります。

## コマンド

すべての vTime コマンドには、使用例と情報が記載された専用ページがあります。詳しくは[コマンドページ](https://vTime.io/commands)をご覧ください。
