# vTime のセットアップ

:::info

vTime をサーバーに招待するには、特定の権限が必要です。これらの権限がないと、Bot をサーバーに追加することはできません。必要な権限は以下の通りです：

- `管理者` 権限
- `サーバー管理` 権限

:::

## ステップ 1: vTime を招待する

[招待リンク](https://vTime.io/invite)を使用して、vTime をサーバーに招待し、アカウントを認証してサーバーを選択します。

## ステップ 2: 認証と権限の付与

vTime を招待すると、アカウントの認証と Bot が正常に機能するために必要な権限の選択を求められます。

## ステップ 3: vTime のロールを設定する

ダッシュボードの一部のモジュールでサーバーのロールを表示するには、vTime のロールをサーバーのロール階層の一番上に移動します。これを行うには：

- サーバー設定に移動
- ロールをクリック
- vTime のロールをクリックしてドラッグし、リストの一番上に移動

:::caution

vTime のロールを一番上に移動しても、Discord がダッシュボードにロールを表示しない場合があります。その場合は、次の手順を試してください：

1. vTime のロールを下にドラッグ
2. 変更を保存
3. vTime のロールを上にドラッグ
4. 再度変更を保存

:::

## サポートが必要ですか？

:::info

vTime のセットアップにさらにサポートが必要な場合は、[サポートサーバー](https://discord.gg/vTime)に参加して助けを求めてください。私たちのチームが喜んでお手伝いします！

:::
