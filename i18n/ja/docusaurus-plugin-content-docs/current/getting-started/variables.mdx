# vTime ダッシュボードの変数

メッセージをよりプロフェッショナルで美しく見せるために、さまざまな変数を使用できます。このガイドでは、ダッシュボードのさまざまなセクションで使用できるすべての変数を紹介します。

## 前提条件

変数を使用する前に、デバイスで Discord の開発者モードを有効にする必要があります。このモードを有効にすると、特定のものをメンションするために必要な ID が提供されます。

## Discord 開発者モードの有効化

> PC の場合

1. Discord を起動
2. `ユーザー設定`をクリック
3. `アプリ設定`までスクロールし、`テーマ`をクリック
4. 開発者モードをオンに切り替え

> 携帯電話の場合

1. Discord を起動
2. `ユーザー設定`をタップ
3. `アプリ設定`までスクロールし、`詳細設定`をタップ
4. 開発者モードをオンに切り替え

開発者モードを有効にしたら、変数を使用する準備が整いました。

## ウェルカムメッセージの変数

以下の変数は、Discord ダッシュボードのウェルカムメッセージセクションで使用できます：

| 変数            |                      説明                       |
| --------------- | :---------------------------------------------: |
| `[user]`        |        新しいメンバーをメンションします         |
| `[inviter]`     |            招待者をメンションします             |
| `[server]`      |             サーバー名を表示します              |
| `[userName]`    |        新しいメンバーの名前を表示します         |
| `[inviterName]` |            招待者の名前を表示します             |
| `[memberCount]` |             メンバー数を表示します              |
| `<@user_id>`    |  ID を使用して特定のユーザーをメンションします  |
| `<#channel_id>` | ID を使用して特定のチャンネルをメンションします |
| `<@&role_id>`   |   ID を使用して特定のロールをメンションします   |

## 自動応答の変数

以下の変数は、Discord ダッシュボードの自動応答セクションで使用できます：

| 変数            |                      説明                       |
| --------------- | :---------------------------------------------: |
| `[user]`        |  トリガーを入力したメンバーをメンションします   |
| `[userName]`    |  トリガーを入力したユーザーの名前を表示します   |
| `[invites]`     |               招待数を表示します                |
| `<@user_id>`    |  ID を使用して特定のユーザーをメンションします  |
| `<#channel_id>` | ID を使用して特定のチャンネルをメンションします |
| `<@&role_id>`   |   ID を使用して特定のロールをメンションします   |

## 埋め込みメッセージの変数

以下の変数は、Discord ダッシュボードの埋め込みメッセージセクションで使用できます：

| 変数            |                      説明                       |
| --------------- | :---------------------------------------------: |
| `[text](link)`  |           ハイパーリンクを作成します            |
| `<@user_id>`    |  ID を使用して特定のユーザーをメンションします  |
| `<#channel_id>` | ID を使用して特定のチャンネルをメンションします |
| `<@&role_id>`   |   ID を使用して特定のロールをメンションします   |

## レベリングシステム

以下の変数は、Discord ダッシュボードのレベリングシステムセクションで使用できます：

| 変数              |               説明               |
| ----------------- | :------------------------------: |
| `[level]`         |   指定されたレベルを表示します   |
| `[user] `         |    ユーザーをメンションします    |
| `[user.username]` |      ユーザー名を表示します      |
| `[user.tag]`      |     ユーザータグを表示します     |
| `[oldLevel] `     | 以前のユーザーレベルを表示します |

上記の変数を使用することで、メッセージをよりプロフェッショナルで魅力的にすることができます。
