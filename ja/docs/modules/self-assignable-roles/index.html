リアクションロール

ワークフローとメッセージスタイルを改善するために使用できるさまざまなオプションで、排他的な役割を設定します。

情報

ここで説明する設定はダッシュボードで管理されます。

  1. ダッシュボードにアクセスしてログインします。
  2. 左側のメニューからサーバーを選択します。
  3. モジュールを選択し、有効になっていることを確認します。

設定

Bot がメッセージにリアクションロールのオプションを適用するには、まず「埋め込みメッセージ」を作成してチャンネルに送信する必要があります。

埋め込みメッセージでは、メッセージの作成と送信方法について説明しています。

  1. メニューから「メッセージを追加」を選択します。
  2. メニューから「編集」を選択します。
  3. 「埋め込み」メニューから作成した埋め込みメッセージを選択します。
  4. 「メッセージを選択」オプションからチャンネルに送信した埋め込みメッセージを選択します。

これで、モジュールのオプションの 1 つに進む準備が整いました。🙋

ユーザーごとの役割制限​

特定のリアクションやインタラクションに対してユーザーが持てる役割の制限。

ボタン

ユーザーがワンクリックで簡単に操作できるユニークな Discord ボタンインタラクション。

  1. 「ボタン」をクリックして希望のボタンの色を選択します。
  2. 「ボタンを追加」を選択してボタンを追加します。
  3. 次に、「設定」ボタンと「削除」ボタンが表示されます。
  4. 「設定」を選択すると、次のオプションが表示されます：

ボタンの絵文字とテキスト

  • ボタンに表示したい絵文字とテキスト。
注意

テキストフィールドは必須です。必ず入力してください。

色

  • いつでも変更できる 4 つの色オプションがあります。

ロールを追加​

ボタンにロールを追加し、ユーザーがクリックしたときにそのロールを受け取るようにするには、クリックしてロールを追加できるプラス記号「+」を探します。

情報

1 つの埋め込みメッセージに最大 25 個のボタンを含めることができます。

セレクトメニュー

使いやすい独自の Discord セレクトメニューで、メンバーがインタラクションし、役割を持つオプションを選択します。

  1. ドロップダウンメニューから「セレクトメニュー」を選択します。
  2. 次に、「オプションを追加」を選択します。
  3. いくつかのオプションが表示され、それぞれの詳細が説明されます：

セレクトメニュープレースホルダー​

役割を選択する前に、セレクトメニューオプションの外側に表示されるテキスト。

絵文字​

各オプションの前に表示したい絵文字。

オプションタイトル​

各オプションに表示したいオプションタイトル。

オプション説明​

各オプションタイトルの下に表示したいオプション説明。オプションの内容を説明するために使用できます。

役割​

ユーザーがセレクトメニューオプションとインタラクションする際に持つ役割を設定するオプション。プラス記号「+」をクリックするとサーバーの役割リストが表示されます。

注意

1 つの埋め込みメッセージに最大 20 個のセレクトメニューオプションを含めることができます。

リアクション

従来のリアクションロールで、メンバーがリアクションをクリックすると役割を受け取ります。

  1. ドロップダウンメニューから「リアクション」を選択します。
  2. ドロップダウンメニューから「絵文字を追加」を選択します。
  3. 通常の絵文字または GIF 絵文字を追加するかどうかを選択します。
  4. プラス記号「+」を使用してそのリアクションの役割を作成します。
注意

サーバーの役割がダッシュボードに表示されない場合は、vTime の役割をサーバーの役割階層で上位に移動します。

役割モード

リアクションロールのモードオプションがいくつかあります。

注意

モードはそのメッセージ内のすべてのオプションに適用されます。

切り替えモード​

ユーザーが役割を持っているかどうかに基づいて役割を追加または削除します。推奨

付与モード​

リアクション/インタラクションが追加または削除されるたびに役割を追加します。ルールの受け入れに最適！

テイクモード​

リアクション/インタラクションが追加または削除されるたびに役割を削除します。

通知の種類

通知の種類を管理するための 2 つのオプションがあります。ユーザーのインタラクションに応答するか、静かに保つかを選択できます。

通知タイプ​

このオプションは、ユーザーが役割を追加、削除、または変更しないときに、インタラクションに応じてユーザーにのみ表示される返信メッセージを送信します。

  1. 最初のオプションは、役割が追加されたときの返信メッセージです。
  2. 2 番目のオプションは、役割が削除されたときの返信メッセージです。
  3. 3 番目のオプションは、ユーザーの役割に変更がない場合のメッセージです。
情報

ボットが役割に言及するには、返信メッセージに {0} を含めます。これは役割を指します。

サイレントタイプ​

ボットからの返信メッセージにユーザーの注意を引かずに静かに保ちたい場合、このオプションは返信メッセージを表示しないために最適です。

重要な注意事項

危険

セレクトメニューやボタンがバックグラウンドにあるが使用していない場合、リアクションが壊れる可能性があります。この問題を解決するには、同じメッセージに追加したが使用していないセレクトメニューやボタンオプションを削除する必要があります。

危険

自己割り当て可能な役割を持つ複数のメッセージを同じ埋め込みメッセージにリンクすることはできません。これを行うと問題が発生する可能性があります。

情報

同じオプションに複数の役割を設定するには、「プレミアムティア 1/2」ティアを購読する必要があります。
