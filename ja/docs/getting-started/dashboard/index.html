ダッシュボード：サーバー管理

ダッシュボードでは、さまざまな機能を簡単に利用できます。設定の変更、モデレーター権限の割り当て、モデレーションコマンドの使用など、さまざまなことが可能です。

ダッシュボードへのアクセス​

ダッシュボードへのアクセスは簡単です。以下の手順で進めてください：

  • ウェブサイトにアクセスし、アカウントにログインします。
  • 同じページにダッシュボードボタンが表示されるので、それをクリックするとダッシュボードにアクセスできます。
注意

サーバーを管理するには、以下のいずれかの権限が必要です：

  1. ADMINISTRATOR
  2. MANAGE_GUILD（または MANAGE_SERVER）
  3. カスタムモデレーターロール

モジュールとコマンドの設定​

ダッシュボードの一部のモジュールには、オン/オフを切り替えるスイッチがあり、有効または無効にすることができます。また、一部のページには、ページ内のすべてのオプションを無効にするトップボタンがあります。さらに、カスタマイズを行うための編集/設定ボタンも用意されています。

各オプションのカスタマイズ内容は以下のとおりです：

  • サーバー設定：設定をデフォルトにリセットします。これにより、以下の設定のみが影響を受けます：
    NotifyChannel（無効）、PrivateNotify（無効）、language（en）、modRoles（空）、UseCommandsChannel（無効）、CommandsChannel（空）。
  • 言語：ボットの言語を変更します。
  • コマンド管理：スイッチをオンにしてチャンネルを選択すると、ボットのコマンドは選択したチャンネルでのみ使用できます。
  • ブラックリスト管理：このセクションでは、リアルタイムでブロックされたメンバーが表示されます。
    「削除」ボタンをクリックすると、ブロックを解除できます。新しいメンバーをブロックする場合は、「ブロック」ボタンをクリックし、理由を入力して、メンバー ID、名前、タグ、またはユーザー名で検索してください。即座にブロックされます。
  • モデレーション管理：モデレーションの設定やコマンドを使用できるようにするためのモデレーター役職を選択します。
注意

サーバーがリストに表示されませんか？
これは、ダッシュボードにフルアクセスするために必要な権限（Administrator、MANAGE_GUILD、またはカスタムモデレーター権限）が不足している可能性があります。
ロール設定で必要な権限が明示的に付与されていることを確認してください。
