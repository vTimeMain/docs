vTime プレミアム

vTime のプレミアムページへようこそ。このページでは、プレミアムボットのプロセス、利用可能な機能、よくある質問について詳しく説明しています。

プレミアム​

含まれている機能は以下の通りです：

  • トップ 3 の音声時間
  • 今日のトップ 3
  • ログ
  • トップ 10 通知
ヒント

プレミアム機能は、BETA 版の間はすべてのユーザーに有効です！

よくある質問​

安定版はいつリリースされますか？
安定版のリリース日はまだ決まっていません。現在、テスト中であり、 バグ修正やプロジェクトの更新を行っています。
プレミアム機能はいつ有料になりますか？
プレミアム機能は、安定版が完成して公開されると有料になります。
