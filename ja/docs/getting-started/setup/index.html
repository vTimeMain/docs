vTime のセットアップ

情報

vTime をサーバーに招待するには、特定の権限が必要です。これらの権限がないと、Bot をサーバーに追加することはできません。必要な権限は以下の通りです：

  • 管理者 権限
  • サーバー管理 権限

ステップ 1: vTime を招待する​

招待リンクを使用して、vTime をサーバーに招待し、アカウントを認証してサーバーを選択します。

ステップ 2: 認証と権限の付与​

vTime を招待すると、アカウントの認証と Bot が正常に機能するために必要な権限の選択を求められます。

注意

ボットが正常に動作するためには、以下の権限が必要です：

  1. VIEW_CHANNEL: ボットがチャンネルを見ることができるようにします。
  2. SEND_MESSAGES: ボットがメッセージを送信できるようにします。
  3. EMBED_LINKS: ボットが埋め込みメッセージを送信できるようにします。
  4. MANAGE_MESSAGES: ボットがメッセージを編集または削除できるようにします。

サポートが必要ですか？​

情報

vTime のセットアップにさらにサポートが必要な場合は、サポートサーバーに参加して助けを求めてください。私たちのチームが喜んでお手伝いします！
