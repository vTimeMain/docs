コマンド

このセクションでは、vTime のコマンド、その使用方法、およびそのコマンドを実行する方法について詳しく説明します。

情報

  • スラッシュコマンドはボットのデフォルトのプレフィックスであるため、すべての使用方法と例はスラッシュコマンド / を使って示されます。

  • [ ] はそのコマンドを参照するために必要ですが、( ) は任意です。

一般​

コマンド説明使用方法
/cmdsコマンドリストへのリンクを送信します/cmds
/help特定のコマンドについての詳細な情報を取得します/help [コマンド名]
/infoメンバーの VC 時間に関する詳細情報を取得します/info (ユーザーまたはユーザー ID)
/leaderboardメンバーのボイスチャンネルのタイムランキングを表示します/leaderboard (ページ)

モデレーション​

コマンド説明使用方法
/blacklistブロックされたメンバーのリストを表示します/blacklist (ページ)
/blockメンバーをボイスチャンネルの時間計算からブロックします/block [ユーザー][理由]
/unblockメンバーをボイスチャンネルの時間計算からアンブロックします/unblock [ユーザーまたはユーザー ID]
/resetメンバーのボイスチャンネル時間をリセットします/reset [ユーザーまたはユーザー ID][理由] [確認が必要]
/resetallサーバー内のすべてのメンバーのボイスチャンネル時間をリセットします/reset all [確認が必要]
/setlanguageサーバーのためにボットの言語を設定します/setlanguage (言語)
/resetsettingsボットの設定をデフォルト設定にリセットします/resetsettings [確認が必要]
重要なモデレーションノート

  • Administrator または Manage_Server の権限を持つユーザー、またはダッシュボードで Mod Roles に割り当てられたユーザーのみがモデレーションコマンドを使用できます。

  • これらのコマンドのいずれかを送信すると、確認メッセージが届きます。この確認は、アクションを適用するために必要です。確認を 15〜30 秒以内に行わないと、確認が無効になります。

次のコマンドのみが確認を必要とします：

  1. /reset
  2. /resetall
  3. /resetsettings
