Comandos

En esta sección, obtendrás una comprensión más profunda de los comandos de vTime, su uso y un ejemplo de cómo poner en acción dichos comandos.

info

  • Dado que los comandos de barra son el prefijo predeterminado del bot, todos los usos y ejemplos se ejemplificarán con el comando de barra /.

  • Los signos [ ] son obligatorios para cada comando que se haga referencia con estos signos, pero los signos ( ) son opcionales.

General​

ComandoExplicaciónUso
/cmdsEnviará el enlace para la lista de comandos/cmds
/helpObtén información detallada sobre un comando específico/help [nombre del comando]
/infoObtén información detallada sobre el tiempo en canal de voz de un miembro/info (usuario o ID de usuario)
/leaderboardMuestra la clasificación de los tiempos de los miembros en el canal de voz/leaderboard (página)

Moderación​

ComandoExplicaciónUso
/blacklistMuestra la lista de miembros bloqueados/blacklist (página)
/blockBloquea a un miembro para que no se calcule su tiempo en el canal de voz/block [usuario][razón]
/unblockDesbloquea a un miembro para que se calcule su tiempo en el canal de voz/unblock [usuario o ID de usuario]
/resetRestablece el tiempo de canal de voz de un miembro/reset [usuario o ID de usuario][razón] [Confirmación requerida]
/resetallRestablece el tiempo de canal de voz de todos los miembros en el servidor/reset all [Confirmación requerida]
/setlanguageEstablece el idioma del bot para este servidor/setlanguage (idioma)
/resetsettingsRestablece la configuración del bot a los valores predeterminados/resetsettings [Confirmación requerida]
Notas Importantes de Moderación

  • Solo los usuarios con permisos de Administrator o Manage_Server, o usuarios asignados a los Mod Roles en el panel de control pueden usar los comandos de moderación.

  • Después de enviar uno de estos comandos, recibirás una confirmación. La confirmación es necesaria para aplicar la acción. Hay un tiempo de espera de 15 a 30 segundos; si no se confirma dentro de ese tiempo, la confirmación será deshabilitada.

Solo los siguientes comandos requieren confirmación:

  1. /reset
  2. /resetall
  3. /resetsettings
