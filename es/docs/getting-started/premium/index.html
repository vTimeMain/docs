vTime Premium

Bienvenido a la página premium de vTime. En esta página, encontrarás una explicación completa de nuestro proceso de bot premium, incluyendo las características disponibles y preguntas frecuentes.

Premium​

Las características incluidas son:

  • Top 3 Tiempo de Voz
  • Top 3 Hoy
  • Logs
  • Notifica Top 10
tip

¡Las características premium están habilitadas para todos los usuarios durante la versión BETA!

Preguntas Frecuentes​

¿Cuándo se lanzará la versión estable?
Aún no hay una fecha de lanzamiento establecida para la versión estable. Actualmente estamos probando, corrigiendo errores y actualizando el proyecto.
¿Cuándo se hará de pago la función premium?
La función premium se hará de pago una vez que la versión estable esté completada y publicada.
