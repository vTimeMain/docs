Configurar vTime

info

Para invitar a vTime a tu servidor, se requieren ciertos permisos. Sin ellos, el bot no podrá ser añadido a tu servidor. Estos permisos son:

  • Administrador Permiso
  • Gestionar Servidor Permiso

Paso 1: Invitar a vTime​

Utilice el enlace de invitación para invitar a vTime a su servidor y autorizar su cuenta, seleccionando su servidor.

Paso 2: Autorización y Concesión de permisos​

Al invitar a vTime, se te pedirá que autorices tu cuenta y selecciones los permisos necesarios para que el bot funcione correctamente.

precaución

Para que el bot funcione correctamente, debe tener los siguientes permisos:

  1. VIEW_CHANNEL: Permite que el bot vea los canales.
  2. SEND_MESSAGES: Permite que el bot envíe mensajes.
  3. EMBED_LINKS: Permite que el bot envíe mensajes incrustados.
  4. MANAGE_MESSAGES: Permite que el bot edite o elimine mensajes.

¿Necesita ayuda?​

info

Si necesitas más ayuda con la configuración de vTime, únete al Servidor de soporte y solicita ayuda. Nuestro equipo estará encantado de ayudarte.
