Introducción

vTime es un bot multipropósito altamente configurable que ofrece funciones como recepción de imágenes, registros detallados, comandos sociales, moderación, roles autoasignables, notificaciones de redes sociales y protección para tu servidor.

Acerca de Nuestros Docs​

Nuestra documentación ofrece explicaciones exhaustivas sobre cómo utilizar y configurar cada una de las funciones de vTime, así como consejos para solucionar cualquier problema que pueda surgir.

Apartados​

Navegue fácilmente a un apartado específico haciendo clic en su nombre en la barra lateral de la izquierda. Cada apartado tiene su propia página de configuración.

Comandos​

Todos los comandos de vTime tienen una página dedicada con ejemplos de uso e información. Echa un vistazo a nuestra página de comandos para más detalles.

Formato​

Esta documentación utiliza el siguiente sistema de codificación por colores:

tip

Indica consejos relacionados con las funciones.

info

Proporciona información relacionada con las características.

precaución

Destaca las precauciones relacionadas con las características.

nota

Muestra notas relacionadas con las características.

peligro

Indica errores conocidos, problemas y cosas a evitar que pueden afectar negativamente al flujo de trabajo.
